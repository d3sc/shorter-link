## About Shorter-link

Shorter Link is useful for shortening links.

## Installation

```
git clone https://github.com/d3sc/shorter-link.git
```

## Requirement:

Shorter link is need `nodejs`.

## Usage
```console
node app -u "https://example.com"
```
 
```
 node app -h
 
 Options
 -h                 help command
 -u <url>           set target url
```
